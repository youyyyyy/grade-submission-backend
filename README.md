# grade-submission-backend

#### About this project

A Spring Boot Project practice from [Udemy](https://www.udemy.com/course/the-complete-spring-boot-development-bootcamp/)

This project covers:

- REST API
- JPA
- Exception handling
- Field validation
- JWT

<br>

#### Run the Code

Compiles and runs the application:

```
mvn spring-boot:run
```
